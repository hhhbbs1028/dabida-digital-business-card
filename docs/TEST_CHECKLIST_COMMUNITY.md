# 커뮤니티 기능 테스트 체크리스트

## 1. 데이터베이스 마이그레이션

### 1.1 Supabase SQL Editor에서 실행
- [ ] `005_community_core.sql` 파일을 Supabase SQL Editor에서 실행
- [ ] 에러 없이 실행되었는지 확인
- [ ] 다음 테이블이 생성되었는지 확인:
  - [ ] `profiles` 테이블에 `display_name`, `bio`, `avatar_url` 컬럼 추가됨
  - [ ] `connections` 테이블 생성됨
  - [ ] `posts` 테이블 생성됨
  - [ ] `conversations` 테이블 생성됨
  - [ ] `conversation_members` 테이블 생성됨
  - [ ] `messages` 테이블 생성됨

### 1.2 RLS 정책 확인
- [ ] `profiles` 테이블: 로그인 유저 모두 조회 가능
- [ ] `connections` 테이블: owner_id만 접근 가능
- [ ] `posts` 테이블: 로그인 유저 모두 조회, 작성자만 수정/삭제
- [ ] `conversations`, `conversation_members`, `messages`: 멤버만 접근 가능

## 10. Smoke Test (최소 기능 확인)

다음 순서로 빠르게 확인:

1. [ ] 커뮤니티 탭 접근
2. [ ] 친구찾기에서 프로필 1개 클릭
3. [ ] 프로필 상세 모달 확인
4. [ ] "메시지 보내기" 클릭
5. [ ] 채팅 탭에서 메시지 1개 전송
6. [ ] 게시판에서 게시글 1개 작성
7. [ ] 파도타기에서 연결 동기화 실행

모든 단계가 에러 없이 완료되면 기본 기능이 정상 작동하는 것으로 판단합니다.
