## 1. 받은 명함 추가 테스트

### 1.1 기본 명함 추가
- [ ] "받은 명함" 탭으로 이동
- [ ] 테스트용 명함 데이터 준비 (예: 이름, 소속, 이메일 등)
- [ ] API를 통해 `createReceivedCard` 호출하여 명함 추가
  ```javascript
  // 브라우저 콘솔에서 테스트
  import { createReceivedCard } from './src/features/contacts/api/contactsApi';
  await createReceivedCard({
    snapshot: {
      display_name: "홍길동",
      headline: "개발자",
      organization: "테스트 회사",
      email: "test@example.com",
      phone: "010-1234-5678"
    }
  });
  ```
- [ ] 명함 리스트에 새 명함이 표시되는지 확인
- [ ] 명함 클릭 시 상세 정보가 오른쪽에 표시되는지 확인

### 1.2 태그와 메모가 있는 명함 추가
- [ ] 태그가 포함된 명함 추가
  ```javascript
  await createReceivedCard({
    snapshot: { display_name: "김철수", organization: "대학교" },
    tags: ["동아리", "개발"],
    memo: "오늘 만난 분"
  });
  ```
- [ ] 태그가 명함 리스트에 표시되는지 확인
- [ ] 메모 요약이 명함 리스트에 표시되는지 확인

## 3. 검색 기능 테스트

### 3.1 이름으로 검색
- [ ] 검색창에 명함의 이름 일부 입력
- [ ] 해당 명함만 필터링되어 표시되는지 확인

### 3.2 소속으로 검색
- [ ] 검색창에 소속명 입력
- [ ] 해당 소속의 명함만 표시되는지 확인

### 3.3 이메일/전화번호로 검색
- [ ] 검색창에 이메일 주소 입력
- [ ] 검색창에 전화번호 입력
- [ ] 각각 올바르게 필터링되는지 확인

### 3.4 메모/태그로 검색
- [ ] 검색창에 메모 내용 입력
- [ ] 검색창에 태그 이름 입력
- [ ] 각각 올바르게 필터링되는지 확인

### 3.5 검색 초기화
- [ ] 검색창 내용 삭제
- [ ] 모든 명함이 다시 표시되는지 확인

## 4. 폴더 기능 테스트

### 4.1 폴더 생성
- [ ] 왼쪽 폴더 목록에서 "+ 폴더 추가" 버튼 클릭
- [ ] 폴더 이름 입력 (예: "동아리")
- [ ] "만들기" 버튼 클릭
- [ ] 폴더 목록에 새 폴더가 추가되는지 확인

### 4.2 폴더 선택
- [ ] "전체" 클릭 → 모든 명함 표시 확인
- [ ] "폴더 없음" 클릭 → 폴더가 없는 명함만 표시 확인
- [ ] 특정 폴더 클릭 → 해당 폴더의 명함만 표시 확인

### 4.3 폴더 이동
- [ ] 명함 선택 후 상세 뷰에서 "편집" 버튼 클릭
- [ ] 폴더 드롭다운에서 다른 폴더 선택
- [ ] "저장" 버튼 클릭
- [ ] 명함이 선택한 폴더로 이동했는지 확인
- [ ] 폴더 목록에서 해당 폴더 클릭 시 명함이 표시되는지 확인

### 4.4 폴더 삭제
- [ ] 폴더에 속한 명함이 있는 상태에서 폴더 삭제
- [ ] 확인 대화상자에서 "확인" 클릭
- [ ] 폴더가 삭제되고 명함들이 "폴더 없음"으로 이동했는지 확인
- [ ] "폴더 없음"에서 해당 명함들이 표시되는지 확인

## 5. 메모 관리 테스트

### 5.1 메모 추가
- [ ] 명함 선택

- [ ] 메모 입력란에 텍스트 입력
- [ ] "저장" 버튼 클릭

### 5.2 메모 수정
- [ ] 기존 메모가 있는 명함 선택
- [ ] "편집" 버튼 클릭
- [ ] 메모 내용 수정
- [ ] "저장" 버튼 클릭
- [ ] 수정된 메모가 표시되는지 확인

### 5.3 메모 삭제
- [ ] 메모가 있는 명함 선택
- [ ] "편집" 버튼 클릭
- [ ] 메모 내용 모두 삭제
- [ ] "저장" 버튼 클릭
- [ ] 메모가 "메모 없음"으로 표시되는지 확인

## 6. 태그 관리 테스트

### 6.1 태그 추가
- [ ] 명함 선택
- [ ] 상세 뷰에서 "편집" 버튼 클릭
- [ ] 태그 입력란에 태그 이름 입력 후 "추가" 버튼 클릭 (또는 Enter 키)
- [ ] 태그가 추가되고 표시되는지 확인
- [ ] 여러 태그 추가 테스트

### 6.2 태그 삭제
- [ ] 태그가 있는 명함 선택
- [ ] "편집" 버튼 클릭
- [ ] 태그 옆의 "×" 버튼 클릭
- [ ] 태그가 삭제되는지 확인

### 6.3 중복 태그 방지
- [ ] 이미 존재하는 태그와 동일한 이름으로 태그 추가 시도
- [ ] 중복 태그가 추가되지 않는지 확인
